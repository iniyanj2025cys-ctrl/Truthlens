## TruthLens

TruthLens is a Flutter web application that analyzes AI-generated responses and presents
confidence indicators, risk labels, and visual graphs to help users interpret AI output responsibly.

## Features
- AI-powered response generation
- Confidence estimation heuristic
- Automatic star-based trust rating
- Risk classification (Low / Instability / Hallucination)
- Line graph & pie chart visualization
- Modern Flutter Web UI

## Tech Stack
- Flutter (Google)
- Dart
- Flutter Web
- fl_chart
